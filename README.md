# Telebot Project

## Требования
- Python 3.10.11
- Docker (опционально)

## Локальная установка

### 1. Создание виртуального окружения

.venv\Scripts\activate

установить пакеты:

 pip install aiogram

 pip install python-dotenv

 pip install aiosqlite

 pip install sqlalchemy

 pip install mistralai


программа запускается 

 python main.py
